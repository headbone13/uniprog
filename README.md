# unicode
Это проект UniProg от ToonVyksa-CODE. Он позволит людям делать привычные вещи через командную строку. В нем возможно будет помощник Юни с ИИ, она будет помогать юзерам с этим кодом и в принципе с Интернетом. Она уже показывает имя вашего компьютера, IP в Интернете и геопозицию по этому самому IP, запускает таймер и калькулятор, который пока может вычесть, перемножить, поделить и сложить только два числа.
